# Emad Afandi Coffee — Website Redesign

A modern ** remake and full redesign ** of the website for Emad Afandi Coffee, a well-known coffee brand in Egypt.  

This project reimagines the brand’s digital presence with a **cleaner interface, improved navigation, bilingual accessibility, and immersive animations** to create a more engaging experience for users exploring the brand and its products.

---

## Project Overview

The goal of this project was to redesign the existing website and build a modern, responsive front-end that better represents the brand’s premium identity while improving usability and performance.

Key focuses included:

- Modernizing the **visual design**
- Improving the **user journey and navigation**
- Enhancing **product discovery**
- Supporting **Arabic and English localization**
- Creating a **smooth and interactive user experience**

---

## ✨ Features

- **Bilingual Support** (Arabic / English) with RTL & LTR layouts  
- **Modern UI/UX Design** with a clean and premium aesthetic  
- **Smooth Animations** and scroll interactions using GSAP  
- **Product Browsing Experience** with category and detail pages  
- **Cart and Checkout Flow**  
- **Fully Responsive Design** for mobile, tablet, and desktop  
- **Fast Development Setup** using Vite
- **More is coming**
